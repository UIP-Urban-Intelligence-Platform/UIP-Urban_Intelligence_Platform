## üéØ Project Overview

**UIP - Urban Intelligence Platform** is a production-ready, multi-agent system for processing real-time traffic data in Ho Chi Minh City and publishing it as **Linked Open Data (LOD)**.

### Key Features

| Feature | Description |
|---------|-------------|
| ü§ñ **41 Agents** | 38 Python + 3 TypeScript AI agents |
| üåê **Full Stack** | React frontend + Express backend |
| üìä **Semantic Web** | NGSI-LD, SOSA/SSN, RDF, SPARQL |
| üê≥ **Cloud Native** | Docker Compose with 12 services |
| üî¨ **Computer Vision** | YOLOX + DETR for traffic analysis |
| üì± **Real-time** | WebSocket + Kafka streaming |
